Comandos de CMD

break:Desactiva la comprobación Ctrl+C en MS-DOS con la que se podían detener procesos en el antiguo sistema operativo.

bitsadmin:Crea y monitorea las cargas y descargas.

call:Llama a un archivo batch dentro de otro archivo batch. El comando no tiene ningún efecto si no se introduce en un archivo batch, sino directamente en CMD.

cd:Muestra el directorio actual y permite cambiar a otros directorios. Con el parámetro /D más la unidad y la ruta también puede cambiarse la unidad.

chcp:Cambia la página de código actual (tabla de códigos) o muestra el número de páginas de la página de código actual.

chdir:Muestra el directorio actual y permite cambiar a otros directorios. Con el parámetro /D más la unidad y la ruta también se puede cambiar la unidad.

choice:Crea una lista de selección: el ejemplo típico es la selección de Sí (S) o No (N) representada como /C SN. Con el parámetro /M puede añadirse un mensaje explicativo para el usuario.

clip:Envía el resultado de un comando al portapapeles. Así se puede, por ejemplo, copiar la estructura de los directorios (dir clip) o el contenido de un archivo (clip < nombre de archivo) en el portapapeles.

cls:Elimina el contenido de la pantalla.

date:Muestra la fecha actual y permite modificarla. Con el parámetro /T solo se muestra la fecha sin opción de modificarla.
