Comandos de Linux

man:El primer comando que presentaré te ayudará a entender todos los demás comandos.

ls:Dentro de una carpeta puedes enlistar todos los archivos que contiene la carpeta usando el comando ls

cd:Una vez que tienes una carpeta, te puedes mover a ella usando el comando cd. cd significa change directory (o cambio de directorio). Se invoca especificando la carpeta a la que te vas a mover. Puedes especificar un nombre de carpeta o una ruta de acceso completa.

pwd:Cuando te sientas perdido en el sistema de archivos, llama al comando pwd para saber dónde estás

mkdir:Puedes crear carpetas usando el comando mkdir

rmdir:Así como puedes crear una carpeta usando mkdir, puedes eliminar una carpeta usando rmdir

mv:Una vez que tienes un archivo, puedes moverlo usando el comando mv. Especificas la ruta actual del archivo, y su nueva ruta

cp:Puedes copiar un archivo usando el comando cp

open:El comando open te permite abrir un archivo usan esta sintaxis

touch:El comando find puede utilizarse para encontrar archivos o carpetas que coincidan con un patrón de búsqueda determinado. Busca de forma recursiva.
